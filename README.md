# Principles-of-Turbulence-with-Python-Companion
Textbook

Principles of Turbulence: An Introduction with Python Computations

Author: Kiran Bhaganagar
University of Texas at San Antonio (UTSA)

This repository contains the companion Python and Jupyter notebooks for the textbook:

Principles of Turbulence: An Introduction with Python Computations

The notebooks are intended to reinforce concepts presented in the text through computation, visualization, and numerical experiments. Readers can reproduce figures, spectra, correlations, transport budgets, and scaling laws discussed throughout the book.

Philosophy

Turbulence is best understood through the combined use of:

Physical intuition
Mathematical analysis
Statistical descriptions
Computational experiments
Data-driven methods

The notebooks are designed not only to reproduce textbook results, but also to encourage exploration and discovery.

Chapter13-Free Shear Flows	Jets, wakes, mixing layers, and plumes
Requirements

Recommended Python version:

Python >= 3.10

Recommended packages:

numpy
scipy
matplotlib
pandas
jupyter
ipywidgets
xarray
h5py

Install packages using:

pip install -r requirements.txt

or

conda env create -f environment.yml
Using the Notebooks

Each notebook is self-contained and may include:

Theory summaries
Numerical examples
Synthetic turbulence generation
Data analysis
Visualization of turbulent structures
Reproduction of textbook figures

Readers are encouraged to modify parameters and perform their own experiments.
